# Duratus Sleep Debt Screener

## Overview
A browser-based tool to assess sleep debt, recovery, and daytime performance. Includes live scoring, RAG classification, and direct PDF export.

## Features
- Card-based questionnaire
- Live scoring dashboard
- RAG classification
- Executive summary report
- Direct PDF export

## Usage
1. Open `index.html`
2. Complete questionnaire
3. Export report as PDF
